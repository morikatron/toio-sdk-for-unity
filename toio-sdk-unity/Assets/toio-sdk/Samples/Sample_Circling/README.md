このサンプルは CubeNavigator のモードの違いによって、多数台のキューブの挙動がどのように変わるか確認するサンプルです。

Unityのインスペクター上で Navi Mode を切り替えることで、集団制御のアルゴリズムを変えることが出来ます。

> 上のgifは Navi Mode を `BOIDS` に指定した場合の挙動です。
